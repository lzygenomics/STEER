# Tutorials

STEER currently documents two tutorial categories in the repository:[^tutorials]

- **Core model pipeline**
- **Generating spliced / unspliced matrices** for different spatial platforms

## Choose a route

### Core pipeline

Use this when your `.h5ad` already contains the layers needed to run STEER.

- [Core Pipeline](core-pipeline.md)

### Preprocessing pipelines

Use one of these when you need to generate `spliced` and `unspliced` matrices first.

- [Slide-seq preprocessing](slide-seq.md)
- [10x Visium preprocessing](visium.md)
- [Stereo-seq preprocessing](stereo-seq.md)

## Recommendation

For most readers, keep the tutorial pages short and practical:

- what data you start from
- what command or notebook to run
- what file you get at the end
- what page to read next
